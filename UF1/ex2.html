HTTP (Protocol d'Transferència d'Hipertexte)
És un protocol de comunicació utilitzat a la World Wide Web per a transferir dades entre un navegador web i un servidor web.
HTML (Llenguatge d'Etiquetatge d'Hipertexte)
És el llenguatge estàndard utilitzat per a crear pàgines web i descriure la seva estructura mitjançant etiquetes i elements.
Navegador
És una aplicació de programari que es fa servir per a accedir i visualitzar pàgines web a la World Wide Web.
URL (Localitzador Uniforme de Recursos)
És una cadena de caràcters que identifica de manera única una adreça d'un recurs a la web. S'utilitza per a accedir a pàgines web i altres recursos en línia.
Client
És un dispositiu o aplicació que fa sol·licituds a un servidor per a accedir a recursos o serveis. En el context d'Internet, un navegador web és un exemple de client.
Servidor
És un dispositiu o programari que respon a les sol·licituds dels clients i ofereix recursos o serveis. En el context web, un servidor web allotja pàgines web i altres continguts.
